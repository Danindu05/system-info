# ⚡ Syfo

A fast and lightweight CLI tool to display detailed system information directly from your terminal.

---

## 🌐 Website Features

* **🖥️ System Overview** – View CPU, memory, and OS details instantly.
* **⚡ Fast & Lightweight** – Runs quickly with minimal resource usage.
* **📊 Clean Output** – Simple and readable terminal display.
* **🧩 Cross Platform** – Works on Linux, macOS, and Windows.
* **🔧 Easy to Use** – Just one command to get all system info.

---

## 🛠️ Technologies Used

* **Node.js**
* **JavaScript**

---


## 💻 Local Development

1. Clone the repository:

   ```sh
   git clone https://github.com/Danindu05/system-info.git

   ```
2. Navigate to the project directory:

   ```sh
   cd system-info

   ```
3. Install dependencies:

   ```sh
   npm install

   ```
4. Run the CLI:

   ```sh
   node index.js
   ```

---

## 📦 Installation

```sh
npm install -g syfo
```

## 🚀 Usage

```sh
syfo
```
